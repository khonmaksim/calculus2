CALCULUS II

QUIZ: Curves in Space and Their Tangents

Find the unit tangent vector and the length of the indicated portion of the curves

Q1. \( \mathbf{r}(t) = (t +1)\hat{i} + (t^2-1)\hat{j}+2t \hat{k}, \quad t =1 \)

Solution for Q1

The position vector is \( \mathbf{r}(t) = (t +1)\hat{i} + (t^2-1)\hat{j}+2t \hat{k} \).

The velocity is: \[ \mathbf{v}(t) = \frac{d}{dt}((t +1)\hat{i} + (t^2-1)\hat{j}+2t \hat{k}) = \hat{i} + 2t \hat{j} + 2 \hat{k}. \]

The magnitude of the velocity is: \[ |\mathbf{v}(t)| = \sqrt{1^2 + (2t)^2 + 2^2} = \sqrt{1 + 4t^2 + 4} = \sqrt{4t^2 + 5}. \]

The unit tangent vector is: \[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{1}{\sqrt{4t^2 + 5}}(\hat{i} + 2t \hat{j} + 2 \hat{k}). \]

The length of the curve from \( t = 0 \) to \( t = 1 \) is: \[ \text{Length} = \int_0^1 |\mathbf{v}(t)| \, dt = \int_0^1 \sqrt{4t^2 + 5} \, dt. \]

The correct answer is: \( T(t) = (1, 2t, 2) \frac{1}{\sqrt{4t^2 + 5}}, \quad \text{Length} = 2.5 \).

Q2. \( \mathbf{r}(t) = (1+ t)\hat{i} + \frac{t^2}{\sqrt{2}}\hat{j}+ \frac{t^3}{3} \hat{k}, \quad t = 1 \)

Solution for Q2

The position vector is \( \mathbf{r}(t) = (1+ t)\hat{i} + \frac{t^2}{\sqrt{2}}\hat{j}+ \frac{t^3}{3} \hat{k} \).

The velocity is: \[ \mathbf{v}(t) = \frac{d}{dt}((1+ t)\hat{i} + \frac{t^2}{\sqrt{2}}\hat{j}+ \frac{t^3}{3} \hat{k}) = \hat{i} + \frac{2t}{\sqrt{2}}\hat{j} + t^2 \hat{k}. \]

The magnitude of the velocity is: \[ |\mathbf{v}(t)| = \sqrt{1^2 + \left(\frac{2t}{\sqrt{2}}\right)^2 + (t^2)^2} = \sqrt{1 + \frac{4t^2}{2} + t^4} = \sqrt{1 + 2t^2 + t^4}. \]

The unit tangent vector is: \[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|}. \]

The length of the curve from \( t = 0 \) to \( t = 1 \) is: \[ \text{Length} = \int_0^1 |\mathbf{v}(t)| \, dt = \int_0^1 \sqrt{1 + 2t^2 + t^4} \, dt. \]

The correct answer is: \( T(t) = (1, \sqrt{2}t, t^2) \frac{1}{t^2 + 1}, \quad \text{Length} = \frac{4}{3} \).

Q3. \( \mathbf{r}(t) = 2 \cos t \hat{i} + 3 \sin t \hat{j} + 4t \hat{k}, \quad t = \pi > 2 \)

Solution for Q3

The position vector is \( \mathbf{r}(t) = 2 \cos t \hat{i} + 3 \sin t \hat{j} + 4t \hat{k} \).

The velocity is: \[ \mathbf{v}(t) = \frac{d}{dt}(2 \cos t \hat{i} + 3 \sin t \hat{j} + 4t \hat{k}) = -2 \sin t \hat{i} + 3 \cos t \hat{j} + 4 \hat{k}. \]

The magnitude of the velocity is: \[ |\mathbf{v}(t)| = \sqrt{(-2 \sin t)^2 + (3 \cos t)^2 + 4^2} = \sqrt{4 \sin^2 t + 9 \cos^2 t + 16}. \]

The unit tangent vector is: \[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|}. \]

The length of the curve from \( t = 0 \) to \( t = \pi \) is: \[ \text{Length} = \int_0^\pi |\mathbf{v}(t)| \, dt. \]

The correct answer is: \( T(t) = (-2 \sin(t), 3 \cos(t), 4) \frac{1}{\sqrt{20 + 5 \cos^2(t)}}, \, \text{Length} = 15.7 \).
