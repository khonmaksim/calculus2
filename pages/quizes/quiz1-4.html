QUIZ: Curvature and Normal Vectors of a Curve

Find the unit tangent vector \( T \), normal vector \( N \), and curvature \( k \).

Q1. \( \mathbf{r}(t) = t \hat{i} + (\ln \cos t) \hat{j}, \quad -\frac{\pi}{2} < t < \frac{\pi}{2} \)

Solution for Q1

The velocity vector is:

\[ \mathbf{v}(t) = \frac{d}{dt} (t \hat{i} + (\ln \cos t) \hat{j}) = \hat{i} - \tan t \hat{j}. \]

The unit tangent vector is:

\[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{(1, -\tan t)}{\sqrt{1 + \tan^2 t}} = (\cos t, -\sin t). \]

The normal vector is:

\[ N(t) = (-\sin t, -\cos t). \]

The curvature is given by:

\[ k = \frac{|T'(t)|}{|v(t)|} = \frac{1}{(1 + \tan^2 t)^{1/2}} = \cos t. \]
Q2. \( \mathbf{r}(t) = (\ln \sec t) \hat{i} + t \hat{j}, \quad -\frac{\pi}{2} < t < \frac{\pi}{2} \)

Solution for Q2

Velocity vector:

\[ v(t) = (\tan t, 1). \]

Unit tangent vector:

\[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{(\tan t, 1)}{\sqrt{1 + \tan^2 t}} = (\sin t, \cos t). \]

Normal vector:

\[ N(t) = (\cos t, -\sin t). \]

Curvature:

\[ k = \frac{|T'(t)|}{|v(t)|} = \cos t. \]
Q3. \( \mathbf{r}(t) = (2t + 3) \hat{i} + (5 - t^2) \hat{j} \)

Solution for Q2

Velocity vector:

\[ v(t) = (2, -2t). \]

Unit tangent vector:

\[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{(2, -2t)}{\sqrt{4 + 4t^2}} = \frac{(1, -t)}{\sqrt{2(1 + t^2)}}. \]

Normal vector:

\[ N(t) = \frac{1}{1+t^2}(-2t, -1+t^2). \]

Curvature:

\[ k = \frac{|T'(t)|}{|v(t)|} = \frac{1}{4(1+t^2)^{3/2}}. \]
