Evaluating Integrals and Solving Initial Value Problems

Q1. Evaluate the integral: \[ \int_0^1 \left( t^3 \hat{i} + 7 \hat{j} + (t+1) \hat{k} \right) \, dt \]

Solution for Q1

The given integral is: \[ \int_0^1 \left( t^3 \hat{i} + 7 \hat{j} + (t+1) \hat{k} \right) \, dt \]

We can break this into components: \[ \int_0^1 t^3 \, dt \hat{i} + \int_0^1 7 \, dt \hat{j} + \int_0^1 (t+1) \, dt \hat{k} \]

For the \( \hat{i} \) component: \[ \int_0^1 t^3 \, dt = \left[ \frac{t^4}{4} \right]_0^1 = \frac{1}{4} \]

For the \( \hat{j} \) component: \[ \int_0^1 7 \, dt = 7 \cdot 1 = 7 \]

For the \( \hat{k} \) component: \[ \int_0^1 (t+1) \, dt = \left[ \frac{t^2}{2} + t \right]_0^1 = \left( \frac{1}{2} + 1 \right) = \frac{3}{2} \]

Therefore, the result of the integral is: \[ \left( \frac{1}{4} \hat{i}, 7 \hat{j}, \frac{3}{2} \hat{k} \right) \]

So, the correct answer is: \( \left( \frac{1}{4} \hat{i}, 7 \hat{j}, 2 \hat{k} \right) \)

Q2. Solve the initial value problem: \[ \frac{dr}{dt} = -t \hat{i} - t \hat{j} - t \hat{k}, \quad r(0) = \hat{i} + 2 \hat{j} + 3 \hat{k} \]

Solution for Q2

The differential equation is: \[ \frac{dr}{dt} = -t \hat{i} - t \hat{j} - t \hat{k} \] Integrating each component with respect to \( t \), we get:

For the \( \hat{i} \) component: \[ \int -t \, dt = -\frac{t^2}{2} + C_1 \]

For the \( \hat{j} \) component: \[ \int -t \, dt = -\frac{t^2}{2} + C_2 \]

For the \( \hat{k} \) component: \[ \int -t \, dt = -\frac{t^2}{2} + C_3 \]

Thus, the general solution is: \[ r(t) = \left( -\frac{t^2}{2} + C_1 \right) \hat{i} + \left( -\frac{t^2}{2} + C_2 \right) \hat{j} + \left( -\frac{t^2}{2} + C_3 \right) \hat{k} \]

Using the initial condition \( r(0) = \hat{i} + 2 \hat{j} + 3 \hat{k} \), we find: \[ C_1 = 1, \quad C_2 = 2, \quad C_3 = 3 \]

Therefore, the solution is: \[ r(t) = \left( 1 - \frac{t^2}{2} \right) \hat{i} + \left( 2 - \frac{t^2}{2} \right) \hat{j} + \left( 3 - \frac{t^2}{2} \right) \hat{k} \]

So, the correct answer is: \( r(t) = (1 - \frac{t^2}{2}) \hat{i} + (2 - \frac{t^2}{2}) \hat{j} + (3 - \frac{t^2}{2}) \hat{k} \)

Q3. Solve the initial value problem: \[ \frac{dr}{dt} = (180t) \hat{i} + (180t - 16t^2) \hat{j}, \quad r(0) = 100 \hat{j} \]

Solution for Q3

The differential equation is: \[ \frac{dr}{dt} = (180t) \hat{i} + (180t - 16t^2) \hat{j} \]

We integrate each component with respect to \( t \):

For the \( \hat{i} \)-component: \[ \int 180t \, dt = 90t^2 + C_1 \]

For the \( \hat{j} \)-component: \[ \int (180t - 16t^2) \, dt = 90t^2 - \frac{16t^3}{3} + C_2 \]

Using the initial condition \( r(0) = 100 \hat{j} \), we find: \[ C_1 = 0, \quad C_2 = 100 \]

Therefore, the solution is: \[ r(t) = 90t^2 \hat{i} + \left( 90t^2 - \frac{16t^3}{3} + 100 \right) \hat{j} \]

So, the correct answer is: \( r(t) = 90t^2 \hat{i} + (90t^2 - \frac{16t^3}{3} + 100) \hat{j} \)
