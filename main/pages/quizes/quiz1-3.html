QUIZ: Arc Length in Space

Find the unit tangent vector and the length of the indicated portion of the curves

Q1. \( \mathbf{r}(t) = 2 \cos t \hat{i} + 2 \sin t \hat{j} + \sqrt{5} t \hat{k}, \quad 0 \leq t \leq \pi \)

Solution for Q1

The position vector is \( \mathbf{r}(t) = 2 \cos t \hat{i} + 2 \sin t \hat{j} + \sqrt{5} t \hat{k} \).

The velocity is: \[ \mathbf{v}(t) = \frac{d}{dt}(2 \cos t \hat{i} + 2 \sin t \hat{j} + \sqrt{5} t \hat{k}) = -2 \sin t \hat{i} + 2 \cos t \hat{j} + \sqrt{5} \hat{k}. \]

The magnitude of the velocity is: \[ |\mathbf{v}(t)| = \sqrt{(-2 \sin t)^2 + (2 \cos t)^2 + (\sqrt{5})^2} = \sqrt{4 \sin^2 t + 4 \cos^2 t + 5} = \sqrt{9} = 3. \]

The unit tangent vector is: \[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{1}{3}(-2 \sin t \hat{i} + 2 \cos t \hat{j} + \sqrt{5} \hat{k}). \]

The length of the curve from \( t = 0 \) to \( t = \pi \) is: \[ \text{Length} = \int_0^\pi |\mathbf{v}(t)| \, dt = \int_0^\pi 3 \, dt = 3\pi. \]

So, the correct answer is: \( T(t) = \frac{1}{3}(-2 \sin t, 2 \cos t, \sqrt{5}), \, \text{Length} = 3\pi \).

Q2. \( \mathbf{r}(t) = 6 \sin 2t \hat{i} + 6 \cos 2t \hat{j} + 5t \hat{k}, \quad 0 \leq t \leq \pi \)

Solution for Q2

The position vector is \( \mathbf{r}(t) = 6 \sin 2t \hat{i} + 6 \cos 2t \hat{j} + 5t \hat{k} \).

The velocity is: \[ \mathbf{v}(t) = \frac{d}{dt}(6 \sin 2t \hat{i} + 6 \cos 2t \hat{j} + 5t \hat{k}) = 12 \cos 2t \hat{i} - 12 \sin 2t \hat{j} + 5 \hat{k}. \]

The magnitude of the velocity is: \[ |\mathbf{v}(t)| = \sqrt{(12 \cos 2t)^2 + (-12 \sin 2t)^2 + 5^2} = \sqrt{144 \cos^2 2t + 144 \sin^2 2t + 25} = \sqrt{169} = 13. \]

The unit tangent vector is: \[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{1}{13}(12 \cos 2t \hat{i} - 12 \sin 2t \hat{j} + 5 \hat{k}). \]

The length of the curve from \( t = 0 \) to \( t = \pi \) is: \[ \text{Length} = \int_0^\pi |\mathbf{v}(t)| \, dt = \int_0^\pi 13 \, dt = 13\pi. \]

So, the correct answer is: \( T(t) = \frac{1}{13}(12 \cos 2t, - 12 \sin 2t, 5), \, \text{Length} = 13\pi \).

Q3. \( \mathbf{r}(t) = t \hat{i} + \frac{2}{3} t^{3/2} \hat{k}, \quad 0 \leq t \leq 8 \)

Solution for Q3

The position vector is \( \mathbf{r}(t) = t \hat{i} + \frac{2}{3} t^{3/2} \hat{k} \).

The velocity is: \[ \mathbf{v}(t) = \frac{d}{dt}(t \hat{i} + \frac{2}{3} t^{3/2} \hat{k}) = \hat{i} + \hat{k} t^{1/2}. \]

The magnitude of the velocity is: \[ |\mathbf{v}(t)| = \sqrt{1^2 + \left(t^{1/2}\right)^2} = \sqrt{1 + t}. \]

The unit tangent vector is: \[ T(t) = \frac{\mathbf{v}(t)}{|\mathbf{v}(t)|} = \frac{1}{\sqrt{1 + t}} (1, 0, t^{1/2}). \]

The length of the curve from \( t = 0 \) to \( t = 8 \) is: \[ \text{Length} = \int_0^8 |\mathbf{v}(t)| \, dt = \int_0^8 \sqrt{1 + t} \, dt. \] The integral gives: \[ \int_0^8 \sqrt{1 + t} \, dt = \frac{2}{3} (1 + t)^{3/2} \Big|_0^8 = \frac{2}{3} [(9)^{3/2} - (1)^{3/2}] = 16. \]

So, the correct answer is: \( T(t) = (1, 0, \frac{1}{3} t), \, \text{Length} = 16 \).
